# YouTube Desktop App

## 概要

YouTube Desktop App は、YouTubeをデスクトップ環境で快適に楽しめるアプリケーションです！  
Google Chromeのタブがどんどん増える日々にさようならを。  
このデスクトップアプリで、あなたの好きな動画体験をもっと便利に、より美しく。  
（ちなみに、作業中にYouTubeを「ちょっとだけ…」が、気付けば数時間経ってる人にもおすすめです😇）

## 特長

- **使いやすいインターフェース**  
  YouTube公式サイトに近しいUIで、初めてでも迷わない設計です。
- **ネイティブアプリの快適さ**  
  ブラウザを閉じても動画視聴は継続！もちろんデスクトップ通知も対応。
- **マルチプラットフォーム対応**  
  Windows／Mac／Linuxで動作します。どのOSでも安心！
- **軽量・高速**  
  Electronベースで無駄のない動作。PCが古くても大丈夫！（多分…）

## インストール方法

1. リリースページから最新のインストーラーをダウンロードしてください。
2. ダウンロードしたファイルを実行します。
3. お好きなフォルダにインストールして完了！

> ※Macの場合は「セキュリティとプライバシー」→「このアプリの実行を許可」でちょっと心のドキドキがあるかもしれませんが、  
> 安心してください。あなたのPCライフを邪魔するものはありません。（むしろ快適さを保証します！）

## 使い方

1. ログイン不要で直接YouTubeにアクセスできます。
2. 動画検索、再生、コメント、チャンネル登録など、基本的な機能は全てそのまま。
3. お気に入りの動画もピン留め可能！

## 開発環境

- **メイン言語**: JavaScript, HTML, CSS
- **フレームワーク**: Electron
- **推奨 Node.js バージョン**: 20.x 以上
- **外部API**: YouTube公式API（※制限あり）
